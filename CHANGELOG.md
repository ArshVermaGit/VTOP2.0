# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-15

### Added

- Initial rewrite of the University Portal.
- Obsidian Glass aesthetic with glassmorphism and ambient lighting.
- Four distinct portals: Student Nexus, Faculty Cabin, Parent Guardian, and Admin Oracle.
- Core academic management features: Attendance, Marks Hub, Time Table, and Course Plan.
- Administrative tools for user provisioning and system monitoring.
- Next.js 16 + React 19 + TypeScript 5.9 tech stack.
- Prisma ORM with PostgreSQL integration.
- NextAuth.js for secure role-based access control.
- Comprehensive project documentation: CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, SUPPORT, ROADMAP, and LICENSE.

### Changed

- Migrated from legacy cluttered interfaces to a clean, focused dashboard.

### Fixed

- Improved load times and performance metrics (Lighthouse 98+).
